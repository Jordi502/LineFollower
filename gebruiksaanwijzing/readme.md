# Gebruiksaanwijzing

### opladen / vervangen batterijen

Opladen:

1. Schakel de robot uit voor het opladen.
2. Gebruik een geschikte 2S Li-ion lader (max. 8,4 V).
3. Laad de batterijen op tot maximaal 4,2 V per cel.
4. Gebruik de robot niet tijdens het opladen.

Vervangen:

1. Zet de knop op de zijkant van de robot op off.
2. Open de behuizing.
3. Verwijder beide 18650-cellen.
4. Plaats twee identieke batterijen (18650).
5. Let op de juiste polariteit.
6. Sluit de behuizing.
7. Knop mag terug op on gezet worden.

### draadloze communicatie
#### verbinding maken

1. Installeer de DSD tech Bluetooth app
2. Open de app en druk op scan.
3. zoek de juiste HM-10 module. (7DF39F51-1FB4-8CF0-62B3-B20D0886EB46)
4. Maak verbinding met de module.
5. Controleer of er verbinding is door in de rechterbovenhoek ‘Disconnect’ te zien.
